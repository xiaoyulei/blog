## 提取一个数二进制位的最后一个1

```n & (~n + 1)```

Example:

```
a = 0b00011100
print(bin(a&(~a+1)))
0b100
```

## 计算数组两个位置之间的中点

一般最简单的写成```M = (L + R) / 2```，但是这里会有个问题，如果数组非常大，```L + R```可能会溢出。

因此可以写成```L + (R - L) / 2```，或者```L + (R - L) >> 1```
